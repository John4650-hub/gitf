##Gitf
This is a utility program to aid in modifying single file in a chosen directory

```bash
gitf --help
usage: gitf [-h] [-t T] [-m M] [-f F] [-p P] [-r R] [-w W]
               [-b B]

This is a utility to make commits to a github repository to
specific files rather than the whole repository

options:
  -h, --help  show this help message and exit
  -t T        your github personal access token
  -m M        Simple message to commit the changes with
  -f F        path to file should be same as the path in your
              github repository
  -p P        whether ypu want to pull a file from github
              repository
  -r R        github repository name
  -w W        owners name
  -b B        branch name
```
